Detailed Description

A MutableDictionaryAttr is a mutable wrapper around a DictionaryAttr.

It provides additional interfaces for adding, removing, replacing attributes within a DictionaryAttr.

We assume there will be relatively few attributes on a given operation (maybe a dozen or so, but not hundreds or thousands) so we use linear searches for everything.
