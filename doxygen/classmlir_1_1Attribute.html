Detailed Description

Attributes are known-constant values of operations and functions.

Instances of the Attribute class are references to immortal key-value pairs with immutable, uniqued key owned by MLIRContext. As such, an Attribute is a thin wrapper around an underlying storage pointer. Attributes are usually passed by value.
