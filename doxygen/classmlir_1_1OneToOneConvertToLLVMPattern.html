Detailed Description

template<typename SourceOp, typename TargetOp>
class mlir::OneToOneConvertToLLVMPattern< SourceOp, TargetOp >

Generic implementation of one-to-one conversion from "SourceOp" to "TargetOp" where the latter belongs to the LLVM dialect or an equivalent.

Upholds a convention that multi-result operations get converted into an operation returning the LLVM IR structure type, in which case individual values must be extracted from using LLVM::ExtractValueOp before being used.
