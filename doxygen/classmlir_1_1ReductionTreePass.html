Detailed Description

template<typename Reducer, TraversalMode mode>
class mlir::ReductionTreePass< Reducer, mode >

This class defines the Reduction Tree Pass.

It provides a framework to to implement a reduction pass using a tree structure to keep track of the generated reduced variants.
