56 /// Creates a pass to convert kernel functions into GPU target object blobs.
57 ///
58 /// This transformation takes the body of each function that is annotated with
59 /// the 'gpu.kernel' attribute, copies it to a new LLVM module, compiles the
60 /// module with help of the GPU backend to target object and then invokes
61 /// the provided blobGenerator to produce a binary blob. Such blob is then
62 /// attached as a string attribute to the kernel function.
63 ///
64 /// Following callbacks are to be provided by user:
65 /// - loweringCallback : lower the module to an LLVM module.
66 /// - blobGenerator : build a blob executable on target GPU.
67 ///
68 /// Information wrt LLVM backend are to be supplied by user:
69 /// - triple : target triple to be used.
70 /// - targetChip : mcpu to be used.
71 /// - features : target-specific features to be used.
72 ///
73 /// Information about result attribute is to be specified by user:
74 /// - gpuBinaryAnnotation : the name of the attribute which contains the blob.
75 ///
76 /// After the transformation, the body of the kernel function is removed (i.e.,
77 /// it is turned into a declaration).
78 std::unique_ptr<OperationPass<gpu::GPUModuleOp>>
79 createConvertGPUKernelToBlobPass(LoweringCallback loweringCallback,
80  BlobGenerator blobGenerator, StringRef triple,
81  StringRef targetChip, StringRef features,
82  StringRef gpuBinaryAnnotation);
