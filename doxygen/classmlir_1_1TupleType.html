Detailed Description

Tuple types represent a collection of other types.

Note: This type merely provides a common mechanism for representing tuples in MLIR. It is up to dialect authors to provides operations for manipulating them, e.g. extract_tuple_element. When possible, users should prefer multi-result operations in the place of tuples.
