static

Check if there are any values used by operations in region defined outside its ancestor region limit.

That is, given A{B{C{}}} with region C and limit B, the values defined in B can be used but the values defined in A cannot. Emit errors if noteLoc is provided; this location is used to point to the operation containing the region, the actual error is reported at the operation with an offending use.
