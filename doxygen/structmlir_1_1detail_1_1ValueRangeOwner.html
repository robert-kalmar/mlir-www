Detailed Description

The type representing the owner of a ValueRange.

This is either a list of values, operands, or an Operation+start index for results.

PointerUnion<const Value *, OpOperand *, void *> mlir::detail::ValueRangeOwner::ptr

The owner pointer of the range.

The owner has represents three distinct states: const Value *: The owner is the base to a contiguous array of Value. OpOperand * : The owner is the base to a contiguous array of operands. void* : This owner is an Operation*. It is marked as void* here because the definition of Operation is not visible here.
