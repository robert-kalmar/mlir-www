Detailed Description

Dialect hooks allow external components to register their functions to be called for specific tasks specialized per dialect, such as decoding of opaque constants.

To register concrete dialect hooks, one should define a DialectHooks subclass and use it as a template argument to DialectHooksRegistration. For example, class MyHooks : public DialectHooks {...}; static DialectHooksRegistration<MyHooks, MyDialect> hooksReg; The subclass should override DialectHook methods for supported hooks.
